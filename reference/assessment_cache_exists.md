# Check if assessment cache exists

Check if assessment cache exists

## Usage

``` r
assessment_cache_exists(end_year, type, level, max_age = 30)
```

## Arguments

- end_year:

  School year end

- type:

  Data type ("tidy" or "wide")

- level:

  Data level

- max_age:

  Maximum age in days (default 30)

## Value

TRUE if valid cache exists
