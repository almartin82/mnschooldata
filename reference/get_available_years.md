# Get available years for enrollment data

Returns the years for which enrollment data is available from MDE.

## Usage

``` r
get_available_years()
```

## Value

Integer vector of available years

## Examples

``` r
get_available_years()
#>  [1] 2007 2008 2009 2010 2011 2012 2013 2014 2015 2016 2017 2018 2019 2020 2021
#> [16] 2022 2023 2024 2025
```
