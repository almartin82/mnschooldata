# Download assessment via MDE portal

Downloads assessment data using the MDEAnalytics portal interface. This
mimics the web interface download functionality.

## Usage

``` r
download_assessment_via_portal(end_year, level, dest_file)
```

## Arguments

- end_year:

  School year end

- level:

  Data level

- dest_file:

  Destination file path

## Value

TRUE if successful, FALSE otherwise
