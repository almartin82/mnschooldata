# Download MDE data via CGI endpoint

Alternative download method using the MDEAnalytics CGI interface. This
mimics the web interface's download functionality.

## Usage

``` r
download_mde_via_cgi(end_year, level, dest_file)
```

## Arguments

- end_year:

  School year end

- level:

  Data level

- dest_file:

  Destination file path

## Value

TRUE if successful, FALSE otherwise
